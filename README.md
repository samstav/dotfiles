dotfiles
========

scripts expect this repo to live in ~/

```bash
cd
git clone https://github.com/samstav/dotfiles
cd dotfiles
./runme.sh
```

Links:
* ~/.bash_aliases
* ~/.bash_profile
* ~/.bashrc
* ~/.gitconfig
* ~/.pystartup
* ~/.tmux.conf
* ~/.vim
* ~/.vimrc

Creates:
~/.autocomplete/git-completion.bash
~/.autocomplete/hg-completion.bash
~/.autocomplete/tmux-completion.bash
